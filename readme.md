#Ezio

## Comment installer Ezio

- Télécharger l'archive zip
- Dézipper l'archive
- Installer un serveur web (simple web server est une bonne solution qui n'oblige pas à télécharger un framework particulier)
- Lancer le serveur en pointant vers le répertoire Ezio
- Se connecter au localhost créé par le serveur web
- Enjoy
